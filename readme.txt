Caldera Forms - UpdraftPlus Integration

== Description ==

This plugin enables automatic upload of forms created in Caldera Forms via UpdraftPlus plugin.
When installed, it add 'UpdraftPlus' processor to Caldera Forms which saves all new forms to uploads/caldera-forms directory (in HTML format).
Processor settings include filename (that can use %field% macros) and HTML header/footer.

The plugin also adds a new entry to the UpdraftPlus backup settings named "Caldera forms".

== Dependencies ==

   * [Caldera Forms](https://wordpress.org/plugins/caldera-forms)
   * [UpdraftPlusPayPal IPN for WordPress](https://wordpress.org/plugins/updraftplus)
